smoothListTriangle¶

Math.smoothListTriangle(datalist, degree=5)¶

Smooth a list of numbers by convolving with a triangle.

The returned data set is shorter by 2*degree-1 samples

:param datalist : data set :type datalist : list of float

:param degree : width of triangle at half maximum in number of samples :type degree : int

Returns

list of smoothed data
