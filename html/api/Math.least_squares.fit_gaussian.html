fit_gaussian¶

Math.least_squares.fit_gaussian(error_function, initial_guess, x, y)¶

Invoke least squares fit to a Gaussian function

:param error_function : function to compute difference between model and data :type error_function : function instance

:param initial_guess : parameters for an initial guess for the parameters :type initial_guess : list of float

:param x : independent variable :type x : numpy array of float

:param y : data :type y : numpy array of float

Returns

(list of parameters, list of formal errors)
